## Supported Formats

1. **PDF** - Portable Document Format with professional styling
2. **DOCX** - Microsoft Word compatible documents
3. **TXT** - Plain text files with metadata
4. **RTF** - Rich Text Format for cross-platform compatibility
5. **HTML** - Web-ready documents with styling
6. **Markdown** - MD format for documentation and notes

## How to Use

1. Open `index.html` in a modern web browser
2. Paste or type your text in the left text box
3. Select your preferred output format from the options
4. Click "Convert & Download" to generate and download your document

## Technical Details

- **Pure HTML/CSS/JavaScript**: No backend required
- **Libraries Used**:
  - jsPDF for PDF generation
  - Native browser APIs for other formats
- **Browser Compatibility**: Works in all modern browsers (Chrome, Firefox, Safari, Edge)

## Installation

Simply download all files and open `index.html` in your browser. No installation or setup required!
